## What It Is

SIGHTLINE is a compact, open-source aim trainer that executes entirely inside a web browser. It is built around the mechanics that matter in tactical shooters, with drills for flick shots, tracking, peeking, and switching between targets. There is nothing to install, no account to create, and no external service to depend on - just open the page and start practicing.

The 2026 release adds more precise difficulty progression plus broader customization controls. You can tune target size, spawn timing, and session length to fit your current level and training goals. Progress is tracked locally through score and accuracy stats, so you can measure improvement over time without sending data to a server. Whether you are warming up before ranked play or isolating a weak point in your aim, SIGHTLINE keeps the focus on practice.

## Script Features

- **Flick Accuracy Drills** - Train fast mouse snaps against static and moving targets

## FAQ

**How do I install SIGHTLINE?**  
There is no install step. Download the HTML file and open it in your browser.

**Will my progress be saved?**  
Session scores are stored locally in your browser's storage. Clearing your browser data will reset them.

**Can I customize the drills further?**  
Yes. Target size, speed, and session length can all be changed from the options menu.

**Does this work with my mouse sensitivity?**  
SIGHTLINE uses raw pointer input and does not modify your system sensitivity. It works with any DPI or in-game sensitivity setting.

**Is this compatible with CS2?**  
Yes. The drills are game-agnostic and designed to improve general aiming mechanics applicable to CS2 and similar titles.

**Where are my training statistics stored?**  
All data is stored locally on your device. No information is sent to any server.
